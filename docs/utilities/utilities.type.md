# Type

## Overview

Sets font-size, font-weight, and line-height together as a single unit.

```html
<p class="u-type-hero">Display text</p>
```
 
| Name            | Description      |
|-----------------|------------------|
| `u-type-hero`   | Hero.            |
| `u-type-4xl`    | 4x large.        |
| `u-type-3xl`    | 3x large.        |
| `u-type-2xl`    | 2x large.        |
| `u-type-xl`     | Extra large.     |
| `u-type-lg`     | Large.           |
| `u-type-md`     | Medium.          |
| `u-type-sm`     | Small.           |
