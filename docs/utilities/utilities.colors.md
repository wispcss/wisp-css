# Color

## Overview

Sets background-color and color together as a single unit.

```html
<div class="u-color-primary"></div>
```

## Colors

| Name                  | Description |
|-----------------------|-------------|
| `u-color-primary`     | Primary.    |
| `u-color-secondary`   | Secondary.  |
| `u-color-neutral`     | Neutral.    |
| `u-color-surface`     | Surface.    |
