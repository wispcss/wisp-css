# Badge

## Overview

Displays a small inline label for status, count, or metadata.

```html
<span class="c-badge">Label</span>
```

## Modifiers

### Shape

| Name               | Description                    |
|--------------------|--------------------------------|
| `c-badge--square`  | Applies a small border radius. |

```html
<span class="c-badge c-badge--square">Label</span>
```
