# Input

## Overview

Applies styles to create an input field.

```html
<input type="text" class="c-input">
```

## Modifiers

### Width

Sets the width of the input.

| Name                   | Description    |
|------------------------|----------------|
| `c-input--width-2ch`   | 2 characters.  |
| `c-input--width-3ch`   | 3 characters.  |
| `c-input--width-4ch`   | 4 characters.  |
| `c-input--width-5ch`   | 5 characters.  |
| `c-input--width-10ch`  | 10 characters. |
| `c-input--width-20ch`  | 20 characters. |
| `c-input--width-30ch`  | 30 characters. |

```html
<input type="text" class="c-input c-input--width-20ch">
```
