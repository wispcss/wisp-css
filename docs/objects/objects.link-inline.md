# Link Inline

## Overview

Inline link pattern for navigation and inline text content.

```html
<a class="o-link-inline">
</a>
```

## Modifiers

### Quiet

Removes underline on interaction.

| Name                   | Description                     |
|------------------------|---------------------------------|
| `o-link-inline--quiet` | Removes the underline on hover. |

```html
<a class="o-link-inline o-link-inline--quiet">
</a>
```
