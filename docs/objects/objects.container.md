# Container

## Overview

Constrains content width and centers it.

```html
<div class="o-container">
</div>
```

<a href="https://codepen.io/wispcss/pen/OPbzPxp" class="wisp-codepen-link">Demo on CodePen</a>

## Modifiers

### Spacing

Applies padding on the block axis.

| Name                        | Description  |
|-----------------------------|--------------|
| `o-container--spacing-xs`   | Extra small. |
| `o-container--spacing-sm`   | Small.       |
| `o-container--spacing-md`   | Medium.      |
| `o-container--spacing-lg`   | Large.       |
| `o-container--spacing-xl`   | Extra large. |

```html
<div class="o-container o-container--spacing-xl">
</div>
```
