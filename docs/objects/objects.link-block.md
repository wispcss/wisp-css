# Link Block

## Overview

Padded link pattern for navigation and tab-style UI.

```html
<a class="o-link-block">
</a>
```

## Modifiers

### Spacing

Applies padding on the inline axis.

| Name                       | Description |
|----------------------------|-------------|
| `o-link-block--spacing-sm` | Small.      |
| `o-link-block--spacing-md` | Medium.     |
| `o-link-block--spacing-lg` | Large.      |

```html
<a class="o-link-block o-link-block--spacing-lg">
</a>
```
