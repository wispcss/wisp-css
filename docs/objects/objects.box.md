# Box

## Overview

Boxes off content from the surrounding elements.

```html
<div class="o-box">
</div>
```

## Modifiers

### Spacing

Applies padding on the block and inline axis.

| Name                  | Description |
|-----------------------|-------------|
| `o-box--spacing-sm`   | Small.      |
| `o-box--spacing-md`   | Medium.     |
| `o-box--spacing-lg`   | Large.      |

```html
<div class="o-box o-box--spacing-lg">
</div>
```
