# Pair

## Overview

Arranges items into two-column key–value pairs with equal row spacing.

```html
<form class="o-pair">
    <label></label>
    <input>
</form>
```

## Modifiers

### Gap

Sets the gap on the inline axis.

| Name                | Description |
|---------------------|-------------|
| `o-pair--gap-sm`    | Small.      |
| `o-pair--gap-md`    | Medium.     |
| `o-pair--gap-lg`    | Large.      |

```html
<form class="o-pair o-pair--gap-sm">
    <label></label>
    <input>
</form>
```
