# Button

## Overview

Applies styles to create a flat button.

```html
<a class="c-button"></a>
```

## Modifiers

### Size

Applies different sizes to the button.

| Name                 | Description |
|----------------------|-------------|
| `c-button--size-sm`  | Small.      |
| `c-button--size-lg`  | Large.      |

```html
<a class="c-button c-button--size-sm"></a>
```

## Notes

You can add both elevation and animations by applying the `press` behavior.

```html
<a class="c-button b-press"></a>
```
